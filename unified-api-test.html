Pixar API Client Test

This page tests the Unified API Client integration with the backend API for Pixar transformations. It checks that the client is correctly sending image and watermark data in the simplified format.

The test will use the production endpoint: https://letzteshemd-faceswap-api-production.up.railway.app

Update: The API has been simplified to only support Pixar transformations. The transformationType and targetImage parameters have been removed from the payload.

Current payload format:

{
  "image": "base64-encoded-image-data",
  "watermark": {
    "url": "watermark-url",
    "width": 200,
    "height": 200,
    "spaceBetweenWatermarks": 100
  }
}
